# OnDemandServicesPlatform_Project
A full-stack web application that connects users with service technicians (Plumbers, Electricians, etc.).
This project features three distinct dashboards for Users, Technicians, and Admins to manage service requests efficiently.

Features
User (Customer)
Register and Login (with AJAX username validation).
Select category, date, time, and describe the issue.
View the status of past requests (Pending, Assigned, Completed).
Update contact information and address.

Technician
Register (Requires Admin Approval).
View jobs assigned by the admin.
Accept jobs and mark them as "Completed" once finished.
View a list of completed tasks.

Admin
View and delete users or technicians.
Verify and approve new technician accounts.
Assign pending user requests to available verified technicians.
View the status of all jobs in the system.

Tech Stack
Frontend: HTML, CSS, JavaScript.
Backend: Native PHP.
Database: MySQL.
Server: Apache (XAMPP).
